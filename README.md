# 🐶 산책관리 시스템 (Dog Walking Management System)

산책관리 시스템은 **견사별 강아지 관리**, **사진 등록/수정**, **산책 기록 조회**, **산책 시작 기능** 등을 제공하는 서비스입니다.  
Express 기반으로 구성되었으며, AWS EC2, S3, PM2 등을 활용하여 서비스 운영 환경을 구축했습니다.

---

## 🚀 Tech Stack

- **Backend**: Node.js (Express)
- **Infra**: AWS EC2
- **Storage**: AWS S3
- **Process Manager**: PM2

---

## 📌 주요 기능 (Features)

### 1. 🐕 견사별 강아지 관리
- 견사(케이지) 별 강아지 목록 조회
- 강아지 추가 / 정보 수정
- 견사 이동 또는 상태 변경 기능

### 2. 📸 사진 등록 및 수정
- AWS S3 업로드 기반 이미지 저장
- 강아지 프로필 사진 등록
- 기존 이미지 교체(수정)

### 3. 🚶 산책하기 (Walk)
- 강아지 산책 시작
- 산책 종료 처리
- 산책 중 상태 관리

### 4. 🗂 산책 기록 관리
- 날짜별 산책 기록 조회
- 산책 시간, 담당자 등의 정보 포함
- 특정 강아지별 기록도 조회 가능
